# CollabCore Doc Editor

Monorepo with:

- `client/`: React + Vite frontend
- `server/`: Node.js + TypeScript + Express + Socket.IO backend

## Requirements

- Node.js (LTS recommended)

## Run locally

### 1) Backend

```bash
cd server
npm install
npm run dev
```

### 2) Frontend

```bash
cd client
npm install
npm run dev
```

## Notes

- Environment variables are expected via `server/.env` (not committed).
